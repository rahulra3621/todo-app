About This ToDo App

Welcome to the ToDo App – your simple and effective task management companion.

Whether you're organizing your day, planning a project, or just staying productive, this app helps you:

  • ✅ Create tasks quickly
  • ✏️ Edit and update existing tasks
  • 🗑️ Delete completed or unnecessary tasks
  • 📆 Track due dates and descriptions

Why We Built It

We believe productivity tools should be clean and hassle-free. This app focuses on a minimal, distraction-free experience so you can stay on track.

How It Works

  • Tasks are stored using a secure database
  • Built with Flask (Python) and HTML/CSS
  • Works in your browser — no setup needed

Future Features

  • 🔔 Reminders and notifications
  • 🗂️ Categories and tags
  • 🌙 Dark mode
  • 🤝 Task sharing

Made with ❤️ as a learning project using Flask.
